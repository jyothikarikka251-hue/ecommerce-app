 E-Commerce Web Application

A full-stack e-commerce web application built using the MERN Stack (MongoDB, Express, React, Node.js).
It provides basic online shopping features like product listing, cart management, and user authentication.

Features
 Product catalog (view products)
 Add / remove items from cart
 User login & registration
 Authentication using JWT
 Backend REST APIs (Products & Orders)
 MongoDB database integration
 Responsive frontend UI
