 E-Commerce Web App

This is a simple full-stack e-commerce web application built using MERN Stack (MongoDB, Express, React, Node.js).

It allows users to browse products, add items to cart, and manage authentication.

 Features
Product listing page
Add to cart & remove from cart
User registration & login
Backend API for products and users
MongoDB database integration
Simple responsive UI
ecommerce-app/
│
├── backend/      (Node + Express + MongoDB)
├── frontend/     (React UI)
├── package.json
├── README.md
How to Run
1. Clone project
git clone https://github.com/your-username/ecommerce-app.git
cd ecommerce-app
2. Install backend dependencies
cd backend
npm install
3. Install frontend dependencies
cd frontend
npm install
4. Start backend
npm start
5. Start frontend
npm start
